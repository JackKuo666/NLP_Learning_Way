# AllenNLP学习笔记
# 1.安装

## 1.1 创建 Conda environment with Python 3.6
```py
    conda create -n allennlp python=3.6
```
## 1.2 打开allennlp环境
```py
    source activate allennlp
```
## 1.3 使用pip安装allennlp依赖和包
```py
pip install allennlp
```
## 测试
直接终端：
```py
$ allennlp
```
或者：
```py
python
import allennlp
```
